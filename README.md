# Telegram-бот для образовательного учреждения Ижевский техникум экономики управления и права

Telegram-бот, созданный для предоставления информации о техникуме и его общежитии. Создан на Python с использованием библиотеки `python-telegram-bot`.

## Функции

- **Интерактивная система меню**: Удобная навигация с помощью клавиатуры
- **Информация об общежитии**:
  - Детали по стоимости.
  - Адрес и местоположение.
  - Вместимость и доступные места.
  - Контактные телефоны.
- **Информация о техникуме**:
  - Часы работы и расписание.
  - Адрес учреждения.
- **Варианты связи**:
  - Номер телефона техникума.
  - Электронный адрес.
  - Ссылки на официальные ресурсы.
- **Контакт разработчика**: Легкий доступ к информации о разработчике бота.

## Необходимые вещи для запуска кода

- **Python 3.8+**
- **python-telegram-bot** (v20.0+)
- **Логирование** для отслеживания ошибок и отладки.

## Установка

1. Скачайте python файл из вкладки **Releases**
